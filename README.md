# 🔐 SecurePass – Advanced Password Generator

An "aesthetic" and high-performance password generator built with **HTML5**, **CSS3**, and **JavaScript**. This project focuses on providing a secure way to generate complex passwords while maintaining a modern, user-friendly "Glassmorphism" interface.



## ✨ Features

- **Customizable Complexity**: Toggle between uppercase letters, numbers, and special symbols.
- **Dynamic Length**: Adjustable slider ranging from 6 to 32 characters.
- **Real-time Strength Meter**: Visual feedback (Weak, Medium, Strong) based on password entropy and length.
- **One-Click Copy**: Integrated Clipboard API with a "Copied!" notification toast.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Shuffled Logic**: Uses a shuffling algorithm to ensure guaranteed characters (like one required number) don't always appear in the same position.

---

## 🎨 Design Philosophy

The UI is built with a **Glassmorphism** aesthetic, featuring:
- Vibrant linear gradients.
- Backdrop blurs and semi-transparent surfaces.
- Modern typography using system-safe sans-serif fonts.
- Interactive components with smooth CSS transitions.
